# 🎨 UI快速测试 - 透明系统栏

## ⚡ 快速验证

### 1. 编译安装（1分钟）
```bash
cd /d c:\freerdp318\client\Android\Studio
gradlew assembleDebug
adb install -r freeRDPCore/build/outputs/apk/debug/freeRDPCore-debug.apk
```

### 2. 视觉检查（30秒）

启动应用后，观察：

✅ **状态栏（顶部）**
- [ ] 可以看到时间
- [ ] 可以看到电量图标
- [ ] 可以看到信号图标
- [ ] 背景是透明的

✅ **导航栏（底部）**
- [ ] 可以看到3个按钮（◁ ○ ▷）
- [ ] 背景是透明的
- [ ] 按钮颜色清晰可见

✅ **内容区域**
- [ ] RDP画面延伸到屏幕边缘
- [ ] 没有黑边

### 3. 功能测试（1分钟）

- [ ] 点击返回键 → 正常工作
- [ ] 点击Home键 → 返回桌面
- [ ] 下拉状态栏 → 显示通知

---

## 📱 不同Android版本效果

| Android版本 | 效果 |
|------------|------|
| 5.0+ | ✅ 完全透明 |
| 4.4 | ⚠️ 半透明（略带黑色） |
| < 4.4 | ❌ 不支持透明 |

---

## 🐛 问题排查

### Q: 系统栏看不见？
**A**: 检查Android版本是否≥4.4

### Q: 内容被系统栏遮挡？
**A**: 正常现象，透明系统栏会覆盖在内容上

### Q: 导航栏不透明？
**A**: 某些定制ROM可能不支持，属于正常
