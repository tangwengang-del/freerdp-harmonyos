================================================================================
                    在 DevEco Studio 中配置签名
================================================================================

您的签名文件已经在 .signing 目录中，现在需要在 DevEco Studio 中配置：

步骤 1：打开签名配置
================================================================================

1. 在 DevEco Studio 菜单中：
   File → Project Structure

2. 在弹出窗口左侧选择：
   Project → Signing Configs

步骤 2：配置签名材料
================================================================================

在右侧面板中配置：

1. 【Support HarmonyOS】选项：勾选

2. 【Signing Configuration】部分：

   Store File(.p12): 
   点击 [...] 按钮，选择：
   c:\huawei\.signing\debug.p12

   Store Password: 
   输入您生成证书时设置的密码
   （如果不记得，可能是：123456 或留空）

   Key Alias: 
   debugKey

   Key Password: 
   输入与 Store Password 相同的密码

   Profile: 
   点击 [...] 按钮，选择：
   c:\huawei\.signing\debug.p7b

   Certpath: 
   点击 [...] 按钮，选择：
   c:\huawei\.signing\debug.cer

   Sign Alg: 
   选择：SHA256withECDSA

3. 点击【Apply】然后【OK】

步骤 3：重新构建
================================================================================

配置完成后：

Build → Build Hap(s) / APP(s) → Build Hap(s)

或按快捷键：Ctrl + F9

步骤 4：如果不知道密码
================================================================================

如果您忘记了生成证书时设置的密码：

方法 1：尝试常见密码
  - 123456
  - 000000
  - 留空（直接不填）

方法 2：重新生成签名文件
  1. 返回华为开发者平台
  2. 删除旧的调试证书
  3. 重新生成新的证书
  4. 下载新的签名文件
  5. 替换 .signing 目录中的文件
  6. 重新配置

================================================================================
                        构建成功后
================================================================================

HAP 文件位置：
  entry\build\default\outputs\default\entry-default-signed.hap

然后就可以上传到服务器测试了！
